About ScuPlan

Project Description

ScuPlan is a professional-grade dive planning application designed for both recreational and technical divers. Built with modern web technologies, it offers comprehensive dive planning, safety calculations, and educational resources.

The application works completely offline once loaded, making it perfect for use on dive boats and remote locations. All calculations follow industry-standard decompression models and safety protocols.

Key Features:

  • ✓ Advanced dive plan calculations with decompression modeling
  • ✓ Technical diving calculators (MOD, END, CNS, Best Mix)
  • ✓ Multi-language support (English, Turkish)
  • ✓ Metric/Imperial unit conversion
  • ✓ Interactive world dive site map
  • ✓ Offline functionality with PWA support
  • ✓ PDF export for dive plans
  • ✓ QR code sharing
  • ✓ Multiple theme support (Light, Dark, Underwater)
